About Coreus

Coreus is a complete UBG with an Ultraviolet proxy, 100 games, tools, that fits in a single HTML file. It was made by n00dle.

Since it is a single HTML file, it is completely unblockable : you can paste the code in a code editor, deploy it using Vercel, Netlify, Surge or any other static host and even convert it in a permanent "data:" URL. See all methods on the GitHub
