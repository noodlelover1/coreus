# Xyzos OS

A modern, web-based operating system built with HTML, CSS, and JavaScript.

## Features

### Bootloader
- **Graphical Boot**: Full GUI boot experience
- **CLI Boot**: Command-line interface boot for advanced users
- **Safe Mode**: Boot with minimal services

### Desktop Environment
- Sleek, modern UI with dark theme
- Taskbar with start menu and system tray
- Desktop icons with drag-and-drop functionality
- Window management (drag, resize, minimize, maximize, close)
- Context menu with desktop actions
- Lock screen with time/date display
- Mobile-responsive design

### Applications
1. **yterminal** - Full-featured command line with 30+ commands
2. **File Manager** - Browse and manage files
3. **Settings** - System configuration
4. **Calculator** - Scientific calculator
5. **Notepad** - Simple text editor
6. **Paint** - Digital drawing app
7. **App Store** - Browse and install apps
8. **Web Browser** - Simple web browser

### Games
1. **Snake** - Classic snake game
2. **Tetris** - Block puzzle game
3. **Flappy Bird** - Endless runner
4. **2048** - Number puzzle game
5. **Minesweeper** - Classic minesweeper
6. **Chess** - Simple chess game
7. **Memory Match** - Card matching game
8. **Tic-Tac-Toe** - Two-player game
9. **Simon Says** - Memory game
10. **Connect 4** - Four in a row
11. **Pong** - Classic paddle game
12. **Whack-a-Mole** - Whack the moles
13. **Sudoku** - Number puzzle

### Terminal Commands
- **System**: help, clear, reboot, shutdown, exit, date, whoami, hostname, uname
- **File**: ls, cd, pwd, cat, mkdir, touch, rm, cp, mv, find
- **Network**: ping, ifconfig, curl, wget
- **Process**: ps, top, kill, history
- **Fun**: matrix, clock, calc, weather, fortune, cow, starwars

## Usage

Simply open `index.html` in a modern web browser.

### Controls
- **Double-click** desktop icons to open apps
- **Click** taskbar apps to switch between windows
- **Right-click** desktop for context menu
- **Click Start** button for start menu

## Technical Details

- Pure HTML/CSS/JavaScript - no frameworks
- LocalStorage for settings persistence
- CSS animations and transitions
- Mobile-responsive design
- Canvas-based games
- No external dependencies
